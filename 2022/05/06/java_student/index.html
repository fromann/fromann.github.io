一、程序简介

1.简介

本程序源码为简化版学生管理系统，仅供学习参考，学生信息存放在ROM中随程序运行而存在，关闭后消失。源码总计300行。所用知识点非list接口

2.功能实现

  • 登记学生信息
  • 显示所有学生信息
  • 根据学号查询学生信息
  • 根据学号插入学生信息
  • 根据学号修改学生信息
  • 根据学号删除学生信息
  • 根据要求排序
