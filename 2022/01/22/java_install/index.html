JAVA环境搭建步骤

小安🎁要加油嗷。

身为JAVA初学者环境搭建一定得看啊！！！

一、下载JDK

进入官网下载JDK安装包JDK（Java SE Development Kit）

二、安装JDK

三、配置Path环境变量

1、配置Path变量
记住以上文件夹目录地址（例如我的：C:\Program Files\Java\jdk-17.0.2）
根据步骤点开高级系统设置
根据以上两张啊图片配置好两个变量
变量：JAVA_HOME 变量值：(一定要设置自己的java目录)C:\Program Files\Java\jdk-17.0.2
在Path 点击编辑添加新环境变量 %JAVA_HOME%\bin

2、验证环境变量

点击（开始+R）打开运行框输入cmd打开windows运行框

依次输入java 和 javac看是否和以下两个界面弹出内容一致若一致则安装完成。

验证java命令（输入java点击回车）
验证javac命令（输入javac点击回车）

NOW java环境搭建完成开始你的学习叭！
