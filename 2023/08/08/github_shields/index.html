Github Shields网站的使用

一、网站简介

这个网站可以实现许多徽章的生成，利于网站搭建过程中徽章的个性化

二、具体操作

1.网站准备

  • 徽章生成站https://www.shields.io/
  • 图标站https://simpleicons.org/

2.徽章生成

shields徽章加入项目状态或者版本等信息，可以访问官网 Shields制作徽章

基本生成方式

根据要求填写内容就可以生成一个简单的徽章啦~
